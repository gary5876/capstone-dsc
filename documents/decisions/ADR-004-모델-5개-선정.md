# ADR-004: 모델 5개 선정

- **일자**: 2026-04-10
- **상태**: 확정

## 배경

"DSC 점수 하락 → 모델 성능 하락"을 입증하려면, 특정 모델에서만 성립하는 게 아니라 다양한 학습 원리의 모델에서 공통적으로 성립해야 한다.

## 결정

| 모델 | 계열 | 오염 민감도 |
|---|---|---|
| LogisticRegression | 선형 | 높음 |
| SVC (linear) | 커널 | 높음 |
| RandomForestClassifier | 배깅 | 낮음 |
| XGBClassifier | 부스팅 | 낮음 |
| MLPClassifier (5층) | 신경망 | 중간 |

## 이유

### 1. 학습 원리가 모두 다름
5개 계열(선형/커널/배깅/부스팅/신경망)을 커버하여 결과의 일반성을 확보.

### 2. 오염 민감도 스펙트럼
- 민감한 모델(LR, SVC): 오염 시 성능 급락 → DSC 하락과 강한 상관
- 강건한 모델(RF, XGB): 오염에 버팀 → "모델에 따라 품질 중요도가 다르다"는 인사이트
- 중간(MLP): 스펙트럼의 중간 지점

### 3. DQ4AI 논문과 비교 가능
DQ4AI가 동일 계열 모델(LR, SVC, MLP, GradientBoosting)로 실험했으므로 결과 교차 검증 가능.

### 4. 실용성
전부 sklearn 기반, GPU 불필요, ~390회 실험이 3~4시간 내 완료.

## 제외된 후보

| 모델 | 제외 이유 |
|---|---|
| KNeighborsClassifier | SVC와 거리 기반 계열 중복 |
| DecisionTreeClassifier | RF의 단일 트리 버전 → RF로 대표 |
| TabNet | pytorch-tabnet 별도 설치 필요, 학습 시간 길어짐 |
| GradientBoosting (sklearn) | XGBoost가 상위 호환 |
