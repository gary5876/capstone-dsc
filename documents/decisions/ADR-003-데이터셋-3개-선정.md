# ADR-003: 데이터셋 3개 선정

- **일자**: 2026-04-10
- **상태**: 확정

## 배경

DSC 검증 실험에 사용할 정형 분류 데이터셋이 필요하다. DQ4AI가 지원하는 데이터셋 중에서 선택해야 polluter 호환이 보장된다.

## DQ4AI 지원 데이터셋 (분류)

| 데이터셋 | 행 | 피처 | 타겟 |
|---|---|---|---|
| SouthGermanCredit | 1,000 | 20 | credit_risk |
| TelcoCustomerChurn | 7,032 | 19 | Churn |
| bank | 39,439 | 3 | education |
| cmc | 1,473 | 9 | Contraceptive method |
| covtype | 581,012 | 54 | class |
| letter | 20,000 | 16 | class (26개) |
| covid | 1,025,152 | 17 | DIED |

## 결정

**TelcoCustomerChurn, SouthGermanCredit, letter** 3개 선정.

## 이유

- **크기 다양성**: 1K / 7K / 20K → 데이터 규모별 영향 확인
- **타겟 다양성**: 이진분류 2개(Telco, Credit) + 다중분류 1개(letter, 26 class)
- **피처 다양성**: 범주 위주(Credit) / 수치+범주 혼합(Telco) / 수치 위주(letter)
- **실용성**: 세 개 합쳐도 수 MB, Colab에서 부담 없음

### 제외 사유

| 데이터셋 | 제외 이유 |
|---|---|
| bank | 피처 3개로 너무 적음 |
| cmc | 1,473행으로 SouthGermanCredit과 크기 중복 |
| covtype | 58만 행으로 실험 시간 과다 |
| covid | 100만 행으로 실험 시간 과다, 의료 도메인 전처리 복잡 |
