**5/5 PASS** (v3.2: 4/5). uniqueness 0.224→0.747, class_balance 0.347→0.834가 가장 큰 회복. v4 가중치가 fitting이 아닌 polluter 무관 일반화 신호임을 정식 검증.

### F2 — 분산 설명력 정직화

- 선형 r² = 0.358
- 비선형 R² (RF 5-fold CV) = **0.632** ± 0.091
- 비율 1.77배

→ DSC 차원에 정보가 충분히 있으나 선형 가중합으로는 일부만 활용. README 한계 섹션에 명시.

### F3 — DSC 정의 회복

- value_accuracy(reference 의존, 0.30) 제거
- label_consistency(0.20) + feature_informativeness(0.10) 신설 = 절대 품질 지표
- baseline 정식 측정값:
  - Telco: LC=0.221, FI=1.000
  - SG: LC=0.077, FI=0.389
  - letter: LC=0.678, FI=1.000

### F4 — 가법 vs min-aggregation

- 가법 (Σwᵢsᵢ): r = 0.598
- min(metrics): r = 0.600
- 차이 -0.002 — 거의 동등. 가법 가정이 본 데이터에서 충분.

### F6 — 모델 클래스별 격차 좁힘

v3.2 모델별 r: 0.31~0.55 (range 0.24)
v4 모델별 r: 0.48~0.70 (range 0.22)

range는 비슷하나 절대값이 모두 +0.11~0.27 향상. 노트북 04에 noise_sensitive / tree_based 프로파일 비교 셀(8.4) 추가 — 두 그룹 모두 프로파일 적용 시 추가 개선.

## 5. 결론

**캡스톤 가설**

> 5종 합성 오염 시나리오에서 정형 분류 데이터에 대한 DSC와 ML F1 macro 사이에 통계적으로 유의한 양의 상관관계 (r=0.598, p<2e-43)가 존재한다.

이 주장이 다음으로 강하게 뒷받침된다:
1. 5/5 polluter hold-out 통과 → fitting 아닌 일반화 신호
2. 3/3 데이터셋 모두 유의 (Telco 회복)
3. 5/5 모델 모두 향상 (range 좁혀짐)
4. 3/3 임계값 세트에서 ANOVA 유의
5. 3/3 ML metric (f1/acc/auc) 모두 유의

자체 인정 한계 (README + 04 §10):
- 합성 오염 한정 (자연 노이즈/시계열/텍스트/이미지 미검증)
- baseline = 원본 가정
- 선형 결합으로 분산의 36%만 설명 (비선형 결합 시 63%)
- 가법성 가정 (차원 간 상호작용 미반영)

캡스톤 발표/논문에서 위 한계를 명시한 상태로 r=0.60, p<1e-43, ANOVA F=84의 강한 통계적 입증 결과 사용 가능.
