## 슬라이드 3 — 결함 두 층위 (1분 20초)

🗣 **"결함을 두 층위로 나눠 진단했습니다. 먼저 데이터 차원, P1과 P2. P1은 outlier_ratio가 가우시안 노이즈에 역반응. 노이즈가 분산을 키우니 IQR도 같이 넓어져 outlier가 줄어든 것으로 측정되는 자기참조 함정. P2는 validity가 타입만 검사해서 가우시안 노이즈를 무감지. letter feature_accuracy 75% 케이스에서 DSC가 베이스라인보다 1.58점 더 올라갔어요. 오염인데 점수가 오른다."**

🗣 **"하지만 P1·P2를 고치고도 더 깊은 메타 논리 결함 10개가 남았습니다. 가장 치명적인 게 F1 — 가중치를 ML 결과를 보고 조정한 게 사실상 fitting이라 입증이 순환 논증이 됐다는 것. F2는 r 0.42여도 r²이 0.18에 불과해 '예측' 주장이 약하다는 것. F3은 value_accuracy가 reference 의존이라 '품질 점수' 정의가 모순됐다는 것."**

📌 두 층위라는 점을 명확히 — 이게 발표의 차별점.

## 슬라이드 4 — 해결: v3.2 → v4 (1분)

🗣 **"v3.2에서는 P1~P7을 처방했습니다. 분포 거리 기반 value_accuracy 신설, outlier가 reference IQR 받게 바꾸고 가중치 재배분. 결과 r이 0.085에서 0.420으로 올라가 1차 입증."**

🗣 **"v4에서는 메타 결함 10개를 처방했습니다. 핵심은 두 가지. 첫째 value_accuracy를 빼고 label_consistency와 feature_informativeness를 신설 — k-NN 라벨 일관성과 mutual information 기반의 절대 품질 지표라 reference 없이도 작동해서 '품질 점수' 정의 회복. 둘째 가중치를 작업 시작 시점에 ADR로 사전 등록하고 ML 결과 본 후 재조정 안 함. 그리고 polluter hold-out 검증으로 사후 확인."**

📌 두 가지 핵심만 또박또박. 디테일은 풀버전.

## 슬라이드 5 — 정식 결과 (1분 30초)

🗣 **"v4 정식 결과입니다. Pearson r이 0.598, p value 1.6 × 10의 마이너스 43승. 등급별 평균 F1도 A 0.88, B 0.78, C 0.60, D 0.52로 깔끔한 단조 감소. ANOVA F 84로 등급 간 차이도 매우 강한 유의."**

🗣 **"가장 중요한 건 polluter hold-out 5/5 PASS입니다. 가장 약했던 uniqueness가 v3.2에서 0.224 FAIL이었는데 v4에서 0.747 PASS로 회복. 이걸로 v4 가중치가 fitting이 아닌 polluter 무관 일반화 신호임이 정량 검증됐습니다."**

🗣 **"또 비선형 R²은 0.63으로 선형 r² 0.36의 1.77배. DSC 차원에 정보가 풍부한데 가중합으로는 일부만 활용한다는 한계도 정직하게 인정."**

📌 산점도, hold-out 막대를 손으로 짚으며 비교.

🗣 **"입증된 주장은 — 5종 합성 오염 시나리오에서 정형 분류 데이터에 대한 DSC와 ML F1 macro 사이에 r 0.598, p value 사실상 0의 양의 상관관계가 존재한다. 다중 정량으로 뒷받침: hold-out 5/5, 데이터셋 3/3 유의, 모델 5/5 향상, 임계값 3/3 sensitivity, ML 지표 3종 모두 r 0.45 초과."**
