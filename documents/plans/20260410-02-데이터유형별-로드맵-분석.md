# 데이터 유형별 확장 설계

> 목적: 정형 이후 텍스트/이미지/음성으로 확장할 때 **무엇을 해야 하는가**를 설계
> 정형은 마스터플랜(20260410-01) 참조. 이 문서는 비정형 유형의 설계만 다룸.

---

## 공통 구조

각 데이터 유형마다 4가지를 만들어야 한다:
1. **오염 방법** — 데이터를 의도적으로 망가뜨리는 코드
2. **DSC 지표** — 품질을 측정하는 코드
3. **모델 학습** — 해당 유형에 맞는 ML/DL 모델
4. **데이터셋** — 공개 데이터셋 확보

> 정형에서 만든 파이프라인(오염 → DSC 계산 → 학습 → 평가 → 스코어보드) 구조를 그대로 재사용하고, 각 유형별로 오염 함수와 DSC 지표만 교체한다.

---

## 1. 텍스트 (Text)

### 오염 방법 (자체 구현)
| 오염 차원 | 방법 | 정형 대응 |
|---|---|---|
| Completeness | 텍스트 필드를 빈 문자열/null로 대체 | 결측치 |
| Uniqueness | 동일 텍스트 행 복제 | 중복 |
| Validity | 랜덤 문자 삽입, 인코딩 깨뜨림 | 형식 오류 |
| Consistency | 같은 의미 다른 표현 혼용 (예: "USA"/"United States"/"미국") | 표현 불일치 |
| Noise | 오타 주입, 단어 순서 섞기 | 피처 노이즈 |
| Label Noise | 감성 라벨 뒤집기 (긍정↔부정) | 라벨 노이즈 |

### DSC 지표 (텍스트 특화)
| 지표 | 계산 방법 |
|---|---|
| Completeness | 빈 텍스트 비율 |
| Uniqueness | 완전 중복 텍스트 비율 |
| Validity | 평균 텍스트 길이 이상치, 인코딩 오류 비율 |
| Consistency | 언어 혼용 비율, 특수문자 패턴 일관성 |
| Class Balance | 라벨 분포 엔트로피 (정형과 동일) |
| Text Quality | 평균 문장 길이 분산, 어휘 다양성 (TTR) |

### 모델 후보
| 모델 | 학습 시간 (Colab GPU) | 비고 |
|---|---|---|
| TF-IDF + LogisticRegression | 수 분 | 베이스라인 |
| BERT-base fine-tuning | 30분~1시간/epoch | 메인 |
| DistilBERT fine-tuning | 15분~30분/epoch | 경량 대안 |

### 데이터셋 후보
| 데이터셋 | 크기 | 태스크 | 출처 |
|---|---|---|---|
| IMDB Reviews | 50K | 감성 분류 | HuggingFace |
| AG News | 120K | 뉴스 분류 (4 class) | HuggingFace |
| NSMC (네이버 영화 리뷰) | 200K | 한국어 감성 분류 | GitHub |

## 2. 이미지 (Image)

### 오염 방법 (자체 구현)
| 오염 차원 | 방법 | 정형 대응 |
|---|---|---|
| Completeness | 이미지 파일 삭제, 손상된 파일 삽입 | 결측치 |
| Uniqueness | 동일 이미지 복제 (정확 복제 + near-duplicate) | 중복 |
| Validity | 해상도 불일치, 채널 수 오류, 잘린 이미지 | 형식 오류 |
| Noise | 가우시안 노이즈, 블러, JPEG 압축 아티팩트 | 피처 노이즈 |
| Class Balance | 특정 클래스 샘플 제거/복제 | 클래스 불균형 |
| Label Noise | 라벨 랜덤 뒤집기 | 라벨 노이즈 |
| Brightness/Contrast | 밝기/대비 극단적 변경 | 이상치 |

### DSC 지표 (이미지 특화)
| 지표 | 계산 방법 |
|---|---|
| Completeness | 손상/누락 이미지 비율 |
| Uniqueness | pHash 기반 near-duplicate 비율 |
| Validity | 해상도/채널/포맷 일관성 |
| Noise Level | 라플라시안 분산 (blur 감지), SNR |
| Brightness | 평균 밝기 분포의 정규성 |
| Class Balance | 라벨 분포 엔트로피 |
| Outlier | 임베딩 공간에서 이상치 거리 |

### 모델 후보
| 모델 | 학습 시간 (Colab GPU) | 비고 |
|---|---|---|
| ResNet-18 fine-tuning | 10~20분/epoch | 베이스라인 |
| ResNet-50 fine-tuning | 20~40분/epoch | 메인 |
| EfficientNet-B0 | 15~30분/epoch | 효율적 대안 |
| ViT-small | 20~40분/epoch | Transformer 계열 |

### 데이터셋 후보
| 데이터셋 | 크기 | 태스크 | 출처 |
|---|---|---|---|
| CIFAR-10 | 60K (170MB) | 분류 (10 class) | torchvision |
| Fashion-MNIST | 70K (30MB) | 분류 (10 class) | torchvision |
| Oxford Flowers-102 | 8K (300MB) | 분류 (102 class) | torchvision |
| 소규모 의료/위성 이미지 | 다양 | 분류 | Kaggle |

## 3. 음성 (Audio)

### 오염 방법 (자체 구현)
| 오염 차원 | 방법 | 정형 대응 |
|---|---|---|
| Completeness | 오디오 파일 삭제/손상, 무음 구간 삽입 | 결측치 |
| Uniqueness | 동일 클립 복제 | 중복 |
| Validity | 샘플레이트 불일치, 채널 수 오류, 클리핑 | 형식 오류 |
| Noise | 백색소음, 환경소음 추가, SNR 조절 | 피처 노이즈 |
| Class Balance | 특정 클래스 제거/복제 | 클래스 불균형 |
| Label Noise | 라벨 뒤집기 | 라벨 노이즈 |
| Duration | 극단적 길이 (너무 짧거나 긴 클립) | 이상치 |

### DSC 지표 (음성 특화)
| 지표 | 계산 방법 |
|---|---|
| Completeness | 손상/누락 파일 비율, 무음 비율 |
| Uniqueness | 오디오 핑거프린트 기반 중복 감지 |
| Validity | 샘플레이트/채널/비트깊이 일관성 |
| SNR | 신호 대 잡음비 분포 |
| Duration | 클립 길이 분포 정규성 |
| Class Balance | 라벨 분포 엔트로피 |

### 모델 후보
| 모델 | 학습 시간 | 비고 |
|---|---|---|
| MFCC + SVM/RF | 수 분 | 전통적 베이스라인 |
| 1D-CNN on spectrogram | 10~20분/epoch | 중간 |
| Whisper fine-tuning | 1시간+/epoch | 최신, 무거움 |
| HuBERT fine-tuning | 40분+/epoch | 음성 특화 |

### 데이터셋 후보
| 데이터셋 | 크기 | 태스크 | 출처 |
|---|---|---|---|
| Speech Commands v2 | 105K (2.3GB) | 키워드 분류 (35 class) | TensorFlow |
| UrbanSound8K | 8.7K (6GB) | 환경음 분류 (10 class) | 공개 |
| ESC-50 | 2K (600MB) | 환경음 분류 (50 class) | GitHub |

## 확장 순서

```
정형 (마스터플랜) → 텍스트 → 이미지 → 음성
```

정형에서 파이프라인 구조를 확립한 뒤, 각 유형은 오염 함수 + DSC 지표 + 모델만 교체하여 확장한다.
