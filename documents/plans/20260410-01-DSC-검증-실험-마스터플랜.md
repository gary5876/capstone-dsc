## DQ4AI Polluter ↔ DSC 지표 매핑

### 사용할 Polluter (5개)

| DQ4AI Polluter | DSC 지표 | 오염 방식 | 강도 조절 |
|---|---|---|---|
| CompletenessPolluter | Completeness (완전성) | 결측치(placeholder) 주입 | 0~100% (5%단위, 21단계) |
| UniquenessPolluter | Uniqueness (유일성) | 행 복제 (duplicate factor) | factor 1.0~5.0 |
| FeatureAccuracyPolluter | Validity (유효성) | 수치: 가우시안 노이즈, 범주: 카테고리 스왑 | 0~100% (5%단위) |
| ConsistentRepresentationPolluter | Consistency (일관성) | 범주값에 "-1","-2" 등 변형 표현 추가 | 0~100% (5%단위) |
| ClassBalancePolluter | Class Balance (클래스 균형) | 클래스 비율 선형 왜곡 | imbalance 0.0~1.0 |

### 사용하지 않는 Polluter

| Polluter | 이유 |
|---|---|
| TargetAccuracyPolluter | 라벨 노이즈는 DSC 지표가 아닌 모델 성능에만 직접 영향 → 별도 분석 가능하지만 핵심 실험에서 제외 |

### DSC 지표 중 Polluter가 없는 것

| DSC 지표 | 대응 방안 |
|---|---|
| **Outlier Ratio** | FeatureAccuracyPolluter의 가우시안 노이즈가 간접적으로 이상치 생성 → 별도 polluter 불필요, DSC 지표로 측정만 |
| **Feature Correlation** | 오염 대상 아님. 원본 데이터의 고유 속성으로 측정만 수행 |

> **실험에서 실제로 "오염 → DSC 변화 → 성능 변화"를 직접 검증하는 차원은 5개.**
> Outlier Ratio, Feature Correlation은 베이스라인 DSC 점수 산출에만 참여.

## DSC 스코어링 엔진

### 7개 지표 (평가지표 가이드 기반)

| 지표 | 가중치(default) | 계산식 | 비고 |
|---|---|---|---|
| Completeness | 0.25 | 1 - (null_count / total_count) | |
| Uniqueness | 0.20 | 1 - (duplicate_count / total_count) | |
| Validity | 0.20 | 타입 일관성 + 형식 유효성 | |
| Consistency | 0.15 | 통과 규칙 수 / 전체 규칙 수 | ⚠️ 아래 참조 |
| Outlier Ratio | 0.10 | 1 - (outlier_count / total) | IQR 기반 |
| Class Balance | 0.05 | 엔트로피 기반 균형 점수 | |
| Feature Correlation | 0.05 | 1 - (고상관 쌍 / 전체 쌍) | threshold 0.95 |

### ⚠️ Consistency 지표 처리 방안

PDF 가이드에서 Consistency는 **LLM이 규칙을 자동 생성**하는 구조이다:
```
LLM 개입 1: 컬럼명 + 샘플 5행 → 규칙 생성 (date_order, numeric_range 등)
LLM 개입 2: 해석 + 개선 가이드
```

DSC 검증 실험에서의 대응:
- **방안 A (권장)**: Consistency 규칙을 **데이터셋별로 수동 정의**. 3개 데이터셋이니 수동으로 충분.
  - 예: TelcoCustomerChurn → "tenure >= 0", "MonthlyCharges > 0" 등
- **방안 B**: LLM API 호출로 자동 생성 (비용 발생, 복잡도 증가)
- **방안 C**: Consistency 지표를 skip하고 6개 지표로 운영 (가중치 재분배)

> ConsistentRepresentationPolluter는 범주값 표현을 변형하므로,
> 규칙 기반이 아닌 **"같은 컬럼에 의미적으로 동일한 값이 다르게 표현된 비율"**로 측정하면
> LLM 없이도 Consistency를 측정할 수 있다.
> → DQ4AI의 compute_quality_measure()를 그대로 활용하는 것이 가장 깔끔.

### DSC 점수 산출
```
DSC Score = Σ(지표 점수 × 가중치) × 100    범위: 0~100
등급: A(90+) / B(75-89) / C(60-74) / D(<60)
프로파일: ml_training / analytics / exploration / default
```

## Phase 3: DQ4AI 오염 실행

### 3-1. 오염 강도 선택

DQ4AI는 5% 단위 21단계를 제공하지만, 전부 쓰면 실험 수가 과다.
**핵심 강도 5단계만 선택:**

| 단계 | 비율 | 의미 |
|---|---|---|
| 0% | 0.0 | 원본 (베이스라인) |
| 10% | 0.1 | 경미한 오염 |
| 25% | 0.25 | 중간 오염 |
| 50% | 0.5 | 심한 오염 |
| 75% | 0.75 | 극심한 오염 |

### 3-2. 오염 실행 (Colab 02)
```python
import sys
sys.path.append('/content/drive/MyDrive/dsc/dq4ai')
from polluters.completeness_polluter import CompletenessPolluter
# ... 각 polluter import

POLLUTION_LEVELS = [0.0, 0.1, 0.25, 0.5, 0.75]

for dataset_name in ['TelcoCustomerChurn', 'SouthGermanCredit', 'letter']:
    df = pd.read_csv(f'/content/drive/MyDrive/dsc/data/raw/{dataset_name}.csv')
    for polluter_cls in [CompletenessPolluter, UniquenessPolluter, ...]:
        for level in POLLUTION_LEVELS:
            # polluter 인스턴스 생성 + 오염 적용
            polluted_df, quality = polluter(df)
            # 저장
            save_path = f'/content/drive/MyDrive/dsc/data/polluted/{dataset_name}/{polluter_name}_{int(level*100)}/'
            polluted_df.to_csv(save_path + 'data.csv', index=False)
```

### 4-2. 학습 루프 (Colab 03)
```python
for dataset_name in DATASETS:
    for polluter_name in POLLUTERS:
        for level in POLLUTION_LEVELS:
            df = load_polluted(dataset_name, polluter_name, level)
            X_train, X_test, y_train, y_test = split(df)
            for model_name, model in MODELS.items():
                model.fit(X_train, y_train)
                y_pred = model.predict(X_test)
                scores = evaluate(y_test, y_pred)
                save_result(dataset_name, polluter_name, level, model_name, scores)
```

## Phase 6: 통계적 검증

### 6-1. 상관관계 분석
- DSC Score ↔ F1-score 간 **피어슨 상관계수** (선형 관계)
- DSC Score ↔ F1-score 간 **스피어만 상관계수** (순위 관계)
- **p-value** < 0.05 (통계적 유의성)
- 목표: r > 0.7

### 6-2. 핵심 주장 정리
```
"DSC Score와 모델 F1-score 간 피어슨 상관계수 r = 0.XX (p < 0.001)"
"DSC 등급 A 데이터의 평균 F1: 0.8X, 등급 D: 0.5X"
"DSC Score가 10점 하락할 때 F1-score 평균 X% 하락"
```
