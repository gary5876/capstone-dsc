# 2026-04-12: DSC 엔진 v2 구현 완료

## 작업 내용

### 1. DSC 엔진 v2 수정사항

| 지표 | v1 (문제) | v2 (수정) |
|---|---|---|
| **Completeness** | `isnull()`만 카운트 → placeholder(-1, "empty") 감지 불가 | placeholder 값도 결측으로 카운트. `compute_dsc()`에 `placeholder_numerical`, `placeholder_categorical` 파라미터 추가 |
| **Consistency** | unique값 수 비교 → 오염 존재 여부만 감지, 양은 무시 | 접미사(-1, -2 등)가 붙은 **행의 비율** 직접 측정. 오염 강도에 비례하여 점수 변화 |
| **Class Balance** | 엔트로피 기반 → 이진분류에서 민감도 부족 | 최소 클래스 비율 / 이상적 균등 비율. 예: 2-class 90:10 → 0.2 |
